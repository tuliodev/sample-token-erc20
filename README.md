# ERC-20 Token sample

Project made to study the ERC-20 token pattern with solidity



# Tools :hammer_and_wrench:
- Hardhat 
- Solidity
- Npm
- Javascript

# Features :rocket:

- Get all token supply
- Get a token balance of a user by your address
- Transfer token between users
